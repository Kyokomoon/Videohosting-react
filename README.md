# Видеохостинг-react

## что бы запустить бек:

- 1 Заходим в папку back
- 2 вводим команду 
```
  python -m venv venv
```
- 3 активируем среду
 ```
  .\venv\Scripts\activate
  ```
- 4 заходим в videohosting
- 5 устанавливаем библиотеки
```
pip install -r requirements.txt
```
- 5 запускаем файл
```
python manage.py runserver
```

## что бы запустить фронт:

- 1 npm install
- 2 npm start
